Версия Python 3.7.5
Для реализация веб приложения использовал фреймворк Flask

Чтобы запустить приложение, нужно запусть файл script.py 
После запуска перейте по адресу http://127.0.0.1:8000/

Функции приложения:
1) Поиск по иднтификатору (первая ссылка вею-странице ). Нужно ввести геономер, после чего выйдет информация о городе.
2) Поиск по двум городам (вторая ссылка на веб-странице). Нужно вввести два города на русском языке, после чего выйдет информация об этих городах.
3) Вывод заданного кол-ва городов (третья ссылка на веб-странице). Нужно ввести число, которое будет равно числу оторбражаемых городов на странице. 
   После чего выйдет список городов


Доп. информация:
В файле back.py описана вся логика программы.
в файле TimeZone.py находится словарь, который нужен, чтобы определить насколько время отличается в двух городах.
в папке templates находятся html шаблоны для каждой страницы.
в файле RU.txt находится вся информация о городах.
